## 2025-01-20 - Global Search Missing Accessibility and Usability Features
**Learning:** `index.html` lacked the `(Press '/')` keyboard shortcut and the `aria-live` screen reader announcement pattern that existed on other pages like `tools.html` and `resources.html`. In addition, `index.html`'s inline CSS lacked standard utility classes like `.sr-only`, which are typically present in external stylesheets but needed locally if the page uses an inline `<style>` block for core definitions.
**Action:** When adding screen reader-only text via JS to pages with heavy inline styles, always ensure the `.sr-only` class is explicitly defined in that page's styles if it doesn't inherit from a global stylesheet correctly. Ensure UX consistency across similar components (like search inputs) across all pages.
