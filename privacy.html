Privacy Policy

How Zero Trust Knowledge Base handles your data and protects your privacy.

Data Collection

Zero Trust Knowledge Base is a static website with no server-side processing. We do not collect, store, or process any personal data.

  • No cookies: This site does not set any cookies.
  • No analytics: We do not use Google Analytics, tracking pixels, or any other analytics service.
  • No forms: We do not collect email addresses, names, or any personal information through forms.
  • No localStorage: We do not store any data in your browser's local storage or session storage.

Hosting

This site is hosted on GitHub Pages. GitHub may collect technical information such as IP addresses in server logs as part of their standard hosting operations. This is governed by GitHub's Privacy Statement.

External Links

This site contains links to external websites (security tools, learning platforms, threat intelligence resources). These external sites have their own privacy policies and data collection practices. We are not responsible for the privacy practices of third-party websites.

All external links open in new tabs with rel="noopener noreferrer" to prevent the destination site from accessing your browsing context on this site.

Security Measures

We implement the following security measures to protect visitors:

  • Content Security Policy (CSP): Restricts resource loading to prevent cross-site scripting (XSS).
  • X-Frame-Options: Prevents this site from being embedded in iframes (clickjacking protection).
  • X-Content-Type-Options: Prevents MIME-type sniffing attacks.
  • Referrer Policy: Controls what referrer information is sent when navigating away from this site.
  • HTTPS: All connections are encrypted via TLS.

GDPR Compliance

Since this site does not collect or process any personal data, GDPR data subject rights (access, rectification, erasure, portability) do not apply. No consent is required as no personal data processing takes place.

Changes to This Policy

Any changes to this privacy policy will be committed to the GitHub repository with a clear commit message. Check the repository history for the full changelog.
